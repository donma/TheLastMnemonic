# TheLastMnemonic

![alt 無效的助憶詞](https://i.imgur.com/ODI4SRl.jpg)


如果你想要自己設計一組 助記詞但是最後一個字卻過不了 MetaMask 或是其他 Web3 錢包

你可以嘗試看看跑起來這專案，這是用 .net8 AOT 寫的，畢竟這跟你的錢有關係，就自己 compile 吧
